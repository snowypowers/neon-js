Module wallet

class Account()¶

This allows for simple utilisation and manipulating of keys without need the long access methods. Key formats are derived from each other lazily and stored for future access. If the previous key (one level higher) is not found, it will attempt to generate it or throw an Error if insufficient information was provided (eg. trying to generate private key when only address was given.) NEP2 <=> WIF <=> Private => Public => ScriptHash <=> Address

Arguments:
  • str (string|object) – WIF/ Private Key / Public Key / Address or a Wallet Account object.

class Balance()¶

object describing the coins found within an Account. Look up various balances through its symbol. For example, NEO or GAS.

Arguments:
  • bal (object) – Balance object as a JSON.
  • bal.net (string) – ‘MainNet’ or ‘TestNet’
  • bal.address (string) – The address of the Account
  • bal.assetSymbols (Array.<string>) – The symbols of the assets available in this Balance
  • bal.assets (object) – The collection of assets in this Balance
  • bal.tokenSymbols (Array.<string>) – The symbols of the tokens available in this Balance
  • bal.tokens (object) – The collection of tokens in this Balance

nep2.encrypt()¶

Encrypts a WIF key using a given keyphrase under NEP-2 Standard.

Arguments:
  • wifKey (string) – WIF key to encrypt (52 chars long).
  • keyphrase (string) – The password will be encoded as UTF-8 and normalized using Unicode Normalization Form C (NFC).
  • scryptParams (scryptParams) – Parameters for Scrypt. Defaults to NEP2 specified parameters.
Returns:

string – The encrypted key in Base58 (Case sensitive).

class Wallet()¶

Wallet class to read and integrate a Wallet file into the library. This class is responsible for ensuring that the Wallet File is read correctly and usable by the library.

Arguments:
  • file (WalletFile) – Wallet file
  • file.name (string) – Name of wallet
  • file.scrypt (WalletScryptParams) – Scrypt parameters
  • file.accounts (Array.<WalletAccount>) – Accounts in wallet
  • file.extra (object) – Extra information of wallet.
