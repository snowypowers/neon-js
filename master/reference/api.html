API¶

Module api

Core¶

claimGas()¶

Perform a ClaimTransaction for all available GAS based on API

Arguments:
  • config (object) – Configuration object.
  • config.net (string) – ‘MainNet’, ‘TestNet’ or a neon-wallet-db URL.
  • config.address (string) – Wallet address
  • config.privateKey (string) – private key to sign with. Either this or signingFunction is required.
  • config.signingFunction (function) – An external signing function to sign with. Either this or privateKey is required.
Returns:

object – Configuration object.
createTx()¶

Creates a transaction with the given config and txType.

Arguments:
  • config (object) – Configuration object.
  • txType (string|number) – Transaction Type. Name of transaction or the transaction type number. eg, ‘claim’ or 2.
Returns:

object – Configuration object + tx
doInvoke()¶

Perform a InvocationTransaction based on config given.

Arguments:
  • config (object) – Configuration object.
  • config.net (string) – ‘MainNet’, ‘TestNet’ or a neon-wallet-db URL.
  • config.address (string) – Wallet address
  • config.privateKey (string) – private key to sign with. Either this or signingFunction is required.
  • config.signingFunction (function) – An external signing function to sign with. Either this or privateKey is required.
  • config.intents (object) – Intents
  • config.script (string) – VM script. Must include empty args parameter even if no args are present
  • config.gas (number) – gasCost of VM script.
Returns:

object – Configuration object.
getBalanceFrom()¶

Helper method to retrieve balance and URL from an endpoint. If URL is provided, it is not overriden.

Arguments:
  • config (object) – Configuration object.
  • config.net (string) – ‘MainNet’, ‘TestNet’ or a neon-wallet-db URL.
  • config.address (string) – Wallet address
  • api (object) – The endpoint API object. eg, neonDB or Neoscan.
Returns:

object – Configuration object + url + balance
getClaimsFrom()¶

Helper method to retrieve claims and URL from an endpoint.

Arguments:
  • config (object) – Configuration object.
  • config.net (string) – ‘MainNet’, ‘TestNet’ or a neon-wallet-db URL.
  • config.address (string) – Wallet address
  • api (object) – The endpoint APi object. eg, neonDB or Neoscan.
Returns:

object – Configuration object + url + balance
makeIntent()¶

Helper method to convert a AssetAmounts object to intents (TransactionOutput[]).

Arguments:
  • assetAmts (object) – Asset Amounts
  • assetAmts.NEO (number) – Amt of NEO to send.
  • assetAmts.GAS (number) – Amt of GAS to send.
  • address (string) – The address to send to.
Returns:

Array.<TransactionOutput> – TransactionOutput
sendAsset()¶

Function to construct and execute a ContractTransaction.

Arguments:
  • config (object) – Configuration object.
  • config.net (string) – ‘MainNet’, ‘TestNet’ or a neon-wallet-db URL.
  • config.address (string) – Wallet address
  • config.privateKey (string) – private key to sign with. Either this or signingFunction is required.
  • config.signingFunction (function) – An external signing function to sign with. Either this or privateKey is required.
  • config.intents (Array.<TransactionOutput>) – Intents.
Returns:

object – Configuration object.
sendTx()¶

Sends a transaction off within the config object.

Arguments:
  • config (object) – Configuration object.
  • config.tx (Transaction) – Signed transaction.
  • config.url (string) – NEO Node URL.
Returns:

object – Configuration object + response
signTx()¶

Signs a transaction within the config object.

Arguments:
  • config (object) – Configuration object.
  • config.tx (Transaction) – Transaction.
  • config.privateKey (string) – private key to sign with.
  • config.publicKey (string) – public key. Required if using signingFunction.
  • config.signingFunction (function) – External signing function. Requires publicKey.
Returns:

object – Configuration object.

NeonDB¶

doClaimAllGas()¶

Perform a ClaimTransaction for all available GAS based on API

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
  • privateKey (string) – Private Key or WIF.
  • signingFunction (function) – Optional async signing function. Used for external signing.
Returns:

Promise.<Response> – RPC response from sending transaction
doMintTokens()¶

Call mintTokens for RPX

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
  • scriptHash (string) – Contract scriptHash.
  • fromWif (string) – The WIF key of the originating address.
  • neo (number) – The amount of neo to send to RPX.
  • gasCost (number) – The Gas to send as SC fee.
Returns:

Promise.<Response> – RPC Response
doSendAsset()¶

Send an asset to an address

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
  • toAddress (string) – The destination address.
  • from (string) – Private Key or WIF of the sending address.
  • assetAmounts (Object) – The amount of each asset (NEO and GAS) to send, leave empty for 0.
  • signingFunction (function) – Optional signing function. Used for external signing.
Returns:

Promise.<Response> – RPC Response
neonDB.getAPIEndpoint()¶

API Switch for MainNet and TestNet

Arguments:
  • net (string) – ‘MainNet’, ‘TestNet’, or custom neon-wallet-db URL.
Returns:

string – URL of API endpoint.
neonDB.getBalance()¶

Get balances of NEO and GAS for an address

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
  • address (string) – Address to check.
Returns:

Promise.<Balance> – Balance of address
neonDB.getClaims()¶

Get amounts of available (spent) and unavailable claims.

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
  • address (string) – Address to check.
Returns:

Promise.<Claim> – An object with available and unavailable GAS amounts.
neonDB.getRPCEndpoint()¶

Returns the best performing (highest block + fastest) node RPC.

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
Returns:

Promise.<string> – The URL of the best performing node.
getTransactionHistory()¶

Get transaction history for an account

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
  • address (string) – Address to check.
Returns:

Promise.<History> – History
getWalletDBHeight()¶

Get the current height of the light wallet DB

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’.
Returns:

Promise.<number> – Current height.

NEP5¶

doTransferToken()¶

Transfers NEP5 Tokens.

Arguments:
  • net (string) – ‘MainNet’, ‘TestNet’ or a custom NeonDB url.
  • scriptHash (string) – Contract scriptHash
  • fromWif (string) – WIF key of the address where the tokens are coming from.
  • toAddress (string) – The address to send the tokens to.
  • transferAmount (number) – Amount to transfer. This number will be divided by 100000000.
  • gasCost (number) – Amount of gas to pay for transfer.
  • signingFunction (function) – Optional external signing function.
Returns:

Promise.<Response> – RPC response
getToken()¶

Get the token info and also balance if address is provided.

Arguments:
  • url (string) – URL of the NEO node to query.
  • scriptHash (string) – Contract scriptHash.
  • address (string) – Address to query for balance of tokens.
Returns:

Promise.<object> – Object containing name, symbol, decimals, totalSupply. balance will be included if address is provided.
