RPC¶

Module rpc

class RPCClient()¶

RPC Client model to query a NEO node. Contains built-in methods to query using RPC calls.

Arguments:
  • net (string) – ‘MainNet’ or ‘TestNet’ will query the default RPC address found in consts. You may provide a custom URL.
  • version (string) – Version of NEO node. Used to check if RPC methods have been implemented. it will default to DEFAULT_RPC found in CONST

  • req.params (Array) – Array of parameters to send.
  • req.id (number) – Unique id to identity yourself. RPC should reply with same id.
